# SeminarioFinal

Repositorio del seminario final: análisis territorial del equilibrio eléctrico (SADI) integrando variables eléctricas y demográficas.  
Incluye datasets agregados, shapefiles de provincias y notebook reproducible en Google Colab.

## Estructura
- `notebooks/`: notebook principal del análisis.
- `datos/`: datasets agregados (demanda, generación, censo).
- `datos/archivos_de_forma/` (o `shapefiles/`): shapefiles de provincias.
